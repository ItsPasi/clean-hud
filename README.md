<p align="center">
    <h1 align="center">Clean HUD</h1>
</p>

A small Fabric client HUD mod for Minecraft 26.1.2 using official Mojang deobfuscated names.

Features:
- Armor Display
- Effect Display
- Item Count Display
- System clock
- Position Adjustable (fixed positions)
- Background Toggle
