    <h1 align="center">Clean HUD</h1>
</p>

A HUD mod that displays armor, effects and an item counter in a vanilla like style. Uses vanilla textures, so using resourcepacks will automatically adapt the appearance.
You are able to adjust positions, as well as the background of certain elements. The settings include options like an adjustable arrow display and a custom item display.

<h1>Features</h1>

- Armor Display
- Effect Display
- Item Count Display
- System clock
- Positions Adjustable (fixed positions)
- Background Toggle
